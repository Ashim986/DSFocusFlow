# Release Notes

## 1.0.1
- Multi-platform Swift Package for macOS/iOS/iPadOS.
- Token system (colors, typography, spacing, radii, shadows) with light/dark themes.
- State-machine utilities for component state and reducers.
- Reusable components: button, card, header, badge, progress ring, form field,
  text field/area, toggle, segmented control, select, list row, section header,
  metric card, toast, alert, empty state.
- Primitives for visualizations: bubble, pointer badge, arrow, curved arrow,
  graph view, tree graph view.
- Pure unit tests across core, state, and components.
- Demo view for primitives in test target.
